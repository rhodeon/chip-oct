# NOTE: 
Only tested on Linux. 

# Requirement: 
SDL2

# Building:
cd src/

make

# Running:
cd bin/ 

./chip-oct (rom_file)

## Running in GUI mode:
./chip-oct

## Opening ROM directly in terminal:
./chip-oct rom_file


# Controls:
ESC - Quit

F1 - Restart game

# Key-Bindings:
