NOTE: Only tested on Linux. 

Requirement: 
SDL2

Building:
Switch to "src" directory and run "make"

Running:
Switch to "bin" directory for the compiled file

To run in GUI mode:
./chip-oct

To open ROM directly in terminal:
./chip-oct rom_name


Controls:
ESC - quit
F1 - reset game

Key-Bindings:
